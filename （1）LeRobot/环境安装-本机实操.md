# 环境安装 · 本机实操

## 阶段 1:装 Miniconda(GUI)
安装包已下到 `C:\Users\Administrator\Downloads\Miniconda3-latest-Windows-x86_64.exe`(清华镜像)。
1. 双击安装,选 ==Just Me==,路径默认(无中文无空格)
2. ==**不要勾** "Add Miniconda3 to my PATH"==(普通 cmd 用不了 conda 就是因为没加 PATH——这是对的)
3. 装完用开始菜单的 ==**Anaconda Prompt**== 敲命令(**不要用普通 cmd!**)

> 关键教训:`'conda' 不是内部或外部命令` = 你在普通 cmd 里。换 Anaconda Prompt 即可。

## 阶段 2:建环境装包(在 Anaconda Prompt 里)
```shell
# ① 建 3.12 环境(注意不是 3.10)并激活
conda create -y -n lerobot python=3.12
conda activate lerobot

# ② pip 换清华源(装包更快)
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# ③ 装 ffmpeg
conda install ffmpeg=7.1.1 -c conda-forge -y

# ④ 装 GPU 版 PyTorch(RTX 4060 → cu124),必须打印 True
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
python -c "import torch; print(torch.cuda.is_available())"

# ⑤ 从下载目录装 lerobot(可编辑 + 可视化/训练依赖)
cd /d "F:\2026\2026-12-dual_arm_som-100\（1）LeRobot下载"
pip install -e ".[dataset_viz,training]"

# ⑥ 验证
python -c "import lerobot; print('lerobot ok')"
lerobot-train --help
python -c "import torch; print('cuda:', torch.cuda.is_available())"
```
> 若 ⑥ 的 cuda 变回 False(pip 装 lerobot 时把 torch 换成 CPU 版):
> ```shell
> pip install --force-reinstall torch torchvision --index-url https://download.pytorch.org/whl/cu124
> ```

## 阶段 3:可视化检查数据(命令写一行!)
```shell
lerobot-dataset-viz --repo-id agibot/a2d_20260619 --root "E:/2026-1-Agi数据/lerobot_a2d_20260619/dataset" --episode-index 0
```
换 `--episode-index 1/2/...` 逐条看 130 条数据质量。

## 本机注意事项
- ==RTX 4060 仅 8GB 显存==:训练 ACT(3×480×640 相机)`--batch_size` 从 ==2 或 4== 起步,别用 8,防 OOM。
- Anaconda Prompt 是 cmd 内核:换盘 `cd /d`,中文路径加引号,==别用 `\` 换行==(用一行)。
- 选官方 HF 版的原因:确定支持 v3.0 数据;Seeed/feetech 硬件那套等接机械臂再单独装。
